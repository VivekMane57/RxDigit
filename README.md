## Overview

RxDigit is a full-stack healthcare platform that digitizes handwritten prescriptions using OCR and NLP technologies. The platform connects Patients, Doctors, and Pharmacies in a single ecosystem for secure prescription management, medicine billing, and digital health records.

## Features

### Patient App

* Secure Registration & Login
* Digital Health Wallet
* Upload Prescription Images
* View Prescription History
* Online Medicine Payment
* Medical Record Management

### Doctor Dashboard

* Create Digital Prescriptions
* Approve Patient Requests
* View Patient History
* Manage Prescriptions

### Pharmacy Dashboard

* Receive Approved Prescriptions
* Medicine Availability Management
* Automated Billing System
* Razorpay Payment Integration
* Invoice Generation

### AI Features

* OCR-Based Prescription Extraction
* NLP-Based Medicine Recognition
* Automated Prescription Digitization
* Medicine Data Structuring

Doctor Dashboard → Backend APIs → MongoDB

Pharmacy Dashboard → Backend APIs → MongoDB

OCR + NLP Engine → Prescription Processing

Razorpay → Payment Processing
